# VideoGame

![screen1](https://github.com/ZagonAb/VideoGame/blob/f70f6ef9c2571708341340abed07abe5b8966f9a/.meta/screenshots/screen.png)

![screen2](https://github.com/ZagonAb/VideoGame/blob/f70f6ef9c2571708341340abed07abe5b8966f9a/.meta/screenshots/screen1.png)

- Un tema para Pegasus Frontend.
- Inspirado en la colección de "ALLGAMES" de Titanius Launcher.
- Opción de búsqueda por letra.
- AL finalizar el video mostrara el boxFront del juego.


## Instalación

[Descarga](https://github.com/ZagonAb/anbernic-theme/archive/refs/heads/main.zip) y extrae el tema a tu [directorio de temas](http://pegasus-frontend.org/docs/user-guide/installing-themes). Luego puede seleccionarlo en el menú de configuración de Pegasus.
